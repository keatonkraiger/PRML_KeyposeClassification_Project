# PRML Keypose Classification Project

This repository contains Keypose Classification code for the PRML term project. The code is organized into two main directories: `tabular` for tabular data classification and `video` for video-based classification. Each directory includes model definitions, training scripts, and utility functions.

You will need to look through the code and README in each directory you choose to work with. Recall, the video-based classification is extra credit and computationally expensive (likely not feasible without a GPU).

## Note

This code is much more dynamic than our previous projects and will require you to read through it and understand the structure. If you find any bugs or have suggestions, please feel free to submit a pull request or post about it on Piazza :)
